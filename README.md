## Building

### Installing Dependencies

```bash 
npm install
```

### Developement Build
Build is platform dependent, possible values are either firefox or chrome for BROWSER field
```bash 
npm run dev BROWSER
```

### Production Build

```bash 
npm run build BROWSER
```
